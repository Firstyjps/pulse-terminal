# @pulse/charts

Lightweight Charts wrappers + reusable chart components. Owned by **Role 4 (Charts Agent)** in [../../AGENTS.md](../../AGENTS.md).

> **Pinned to lightweight-charts v4.2** — DO NOT upgrade to v5 (breaking API change: removes `addCandlestickSeries` / `addHistogramSeries`).

## Public API
```tsx
<Candlestick data={...} symbol="BTCUSDT" height={400} />
<FlowChart data={...} type="stablecoin" />
<Sparkline data={...} positive width={80} height={24} />
```
